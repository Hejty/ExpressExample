# Express API + frontend example

Make an application that will load tasks.json

Create a simple API:

```
GET /task
GET /task/:id
POST /task
PUT /task/:id
DELETE /task/:id
```

Make a frontend for it

## Setup

Make sure to install dependencies:

```GIT
npm install
```

To compile, use:

```GIT
npx ts-node src/index.ts
```
